# Nxtlookc

Background and certificate verification platform landing page.

## Stack

- React 19 + Vite 7
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Contact form

Submissions go to Formspree (`FORMSPREE_ENDPOINT` in `src/pages/landingpage.jsx`). Direct email: hello@nxtlookc.com.
